🧪 Manual Testing Checklist - Cube Notary Website

1. 📱 Responsive Design Test

Test: Open the main website and resize your browser window

Expected: Layout should adapt smoothly to different screen sizes

Check: Navigation should stack on mobile, buttons should remain clickable

→ Open Main Website

2. 📅 Calendar Functionality Test

Test: Click on any date in the calendar

Expected: Booking modal should open

Check: Modal should display current month with clickable dates

→ Test Calendar

3. 📝 Booking Form Test

Test: Fill out the booking form and submit

Expected: Form validation should work, success message should appear

Check: All required fields should be validated

→ Test Booking Form

4. 📞 Contact Links Test

Test: Click "Call" and "Text" buttons

Check: Links should have correct tel: and sms: protocols

→ Test Contact Buttons

5. 🎨 Visual Design Test

Test: Check overall appearance and styling

Expected: Dark theme, blue accents, professional look

Check: Colors should match brand, text should be readable

→ Check Visual Design

6. ⌨️ Keyboard Navigation Test

Test: Use Tab key to navigate through the page

Expected: All interactive elements should be focusable

Check: Focus indicators should be visible

→ Test Keyboard Navigation

7. 🔍 SEO Elements Test

Test: View page source and check meta tags

Expected: Title, description, and structured data should be present

Check: All SEO elements should be properly formatted

→ Check SEO Elements

8. 🚀 Performance Test

Test: Check page load speed and responsiveness

Expected: Page should load quickly, interactions should be smooth

Check: No lag when clicking buttons or opening modals

→ Test Performance

9. 📱 Mobile Device Test

Test: Open on mobile device or use browser dev tools mobile view

Expected: Layout should be mobile-friendly

Check: Text should be readable, buttons should be touch-friendly

→ Test Mobile View

10. 🔗 External Links Test

Test: Click on Yelp review link in footer

Expected: Should open in new tab

Check: Link should work and open external site

→ Test External Links

📋 Testing Notes

All tests should pass for production deployment.

If any test fails, note the issue and fix before deploying to Render.

Pay special attention to mobile responsiveness and form functionality.

⚠️ Known Issues to Check

• Calendar should show current month with all dates available

• Modal should close when clicking outside or pressing Escape

• Form should validate all required fields

• Contact buttons should work on mobile devices
